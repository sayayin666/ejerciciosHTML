aprendiendo HTML

HTML es un lenguaje de marcado de hipertexto, que es el lenguaje fundamental que se utiliza para estructurar y organizar el contenido de las paginas web.

  1. primer paso: abre tu editor de codigo preferido
  2. segundo paso: crea un archivo de HTML seguido del nombre + un "." e implicita y seguidamente "html"
  3. tercer paso: crea la estructura básica de HTML unsando la tecla (shift) + 1 en tu teclado numérico que está encima de tu teclado de letras

la etiqueta "(link)" en HTML se usa para establecer una relación entre el documento actual y un recurso externo, como una hoja de estilos CSS o un favicon. Se coloca en la seccion "(head)" de la pagina y utiliza el atributo "href" para indicar la ubicaciín del recurso relacionado.


Uso común: Enlazar hojas de estilo CSS: Se usa para vincular una hoja de estilo externa a la página, lo que permite aplicar estilos a los elementos de la página. Añadir un favicon: Se utiliza para definir la imagen que se mostrará como icono de la página en la barra de pestañas del navegador.
